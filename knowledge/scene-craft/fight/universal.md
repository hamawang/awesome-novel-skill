# 战斗/对抗场景写法

> 每条技法以 `[标签]` 开头。prompt-crafter 根据场景核心事件匹配标签，只选取对应的技法注入 L8。

## 标签选择指南

| 场景特征 | 匹配标签 | 选取理由 |
|---------|---------|---------|
| 角色使用重武器/力量型/刚猛风格 | [动词] [反馈] | 用精确动词替代"打踢捶"，用环境反馈表现力量 |
| 角色使用轻武器/敏捷型/刺客风格 | [动词] [感官] | 用"抹挑钻绕"替代"躲闪划"，多感官营造紧张 |
| 需要表现攻击的威力/伤害 | [反馈] [后果] | 不靠形容词自夸，靠受击反馈+环境破碎 |
| 战斗节奏单一、读者看累 | [节奏] [感官] | 长短句交替、三景切换、多感官轰炸 |
| 双方无脑对轰、没有策略 | [博弈] | 加战术动机——抓破绽/卖破绽/借势 |
| 想写"快"（刀快/剑快/身法快） | [动词] [反馈] | 用结果反写速度，不写"快如闪电" |
| 想写"重"（拳重/锤重/碾压） | [反馈] [后果] | 用人体反馈+地面破碎写重量 |

---

## [动词] 精细动词替换——告别广播体操

**核心：** 按角色武风绑定专属动词，禁用通用词。

### 刚猛型（力量、重武器、肉身霸体）

| ❌ 禁用 | ✅ 必用 |
|---------|--------|
| 打、踢、捶、撞 | 轰、劈、砸、崩、碾、凿、撞、拍 |

```
❌ 他用力一拳打在对方胸口。

✅ 他五指骤然攥紧，指骨爆起脆响，拳锋碾碎气流，
   如重锤轰然轰入对方胸腔，劲势透体，
   震得对手后背衣料成片碎裂。
```

### 灵动型（速度、身法、短刃）

| ❌ 禁用 | ✅ 必用 |
|---------|--------|
| 躲、闪、划、打 | 抹、挑、钻、绕、掠、贴、滑、旋 |

**硬性规则：** 所有攻击必须携带发力细节——攥骨、拧腰、踏地、沉肩。禁止无过程直接出招。

---

## [反馈] 力量不靠吹——靠环境+受击反馈

**核心：** 力量 = 后果反馈，不是自夸。不要写"这一拳威力巨大"，写它造成了什么。

### 万能力量公式

```
攻击动作 + 受击者形变 + 环境破碎 + 气浪/余波
```

```
❌ 这一拳威力巨大、万斤巨力。

✅ 三套反馈选至少一套：

人体反馈：胸腔凹陷、气血翻腾、骨骼闷响、身形僵直
地面反馈：地砖龟裂、尘土炸开、脚下塌坑
空气反馈：气浪白芒、破空爆鸣、风压扑面
```

### 顶级反写（写快、写重、写利）

```
写刀快：收刀归鞘，人走十步，身后落叶齐齐断落。

写拳重：拳未至，风压先至，土墙压出清晰拳印。
```

**禁止：** 威力极大、速度极快、力气很重、快如闪电——全是形容词空话。

## [感官] 多感官轰炸——告别只有画面的战斗

**核心：** 战斗不是在看电影，是亲历现场。每场交战至少切换 2 种感官。

### 固定素材库

| 感官 | 可用元素 |
|------|---------|
| **听觉** | 拳风爆鸣、骨骼脆响、刀锋摩擦闷嘶声、兵刃炸响、急促喘息 |
| **嗅觉** | 血腥味、铁锈味、硝烟味、尘土焦糊味 |
| **触觉（体感）** | 虎口震麻、掌心刺痛、肌肉撕裂酸胀、风压压得胸腔发闷、汗水刺眼 |

**规则：** 一场交战至少覆盖 2 种感官，不能只有视觉。

## [节奏] 电影级节奏控制

**核心：** 全程平铺读者犯困。两套铁律控制心跳。

### 铁律 1：长句慢、短句快

| 阶段 | 句型 | 效果 |
|------|------|------|
| 对峙铺垫 | 长句 | 压迫铺垫、环境、眼神、气场 |
| 交手瞬间 | 3-6 字超短句连炸 | 心跳加速、紧张 |

```
对峙：他慢慢压低重心，脚掌在碎石上碾了半圈，盯着对方肩膀的起伏。
交手：身形掠出、踏地崩尘、旋身、斩落、寒光炸现、血花飞溅。
```

### 铁律 2：运镜三切换

```
远景（全景战场）→ 中景（两人交锋）→ 特写（血珠/刀锋火星/指节发白）
```

### 特写素材库

```
血珠悬空、刀锋火星、指节发白、瞳孔骤缩、衣角撕裂、发丝乱飞
```

## [博弈] 动作戏博弈感——告别无脑互殴

**核心：** 每一招都有目的，不是乱打。好看的打斗 = 智商碾压 + 预判博弈 + 绝境利用。

### 三大战术写法

| 战术 | 说明 | 效果 |
|------|------|------|
| **抓破绽** | 盯对方重心、旧伤、落脚错位、失衡瞬间 | 读者觉得主角聪明 |
| **卖破绽** | 故意漏空，引诱对方突进再反杀 | 反转爽感 |
| **借势** | 沙尘、地形、坡度、障碍物、对手性格惯性 | 战斗不单调 |

```
❌ 双方对轰了十几招，不分胜负。

✅ 他退了半步，左脚在碎石上滑了一下——重心不稳。
   对方果然抓住这个机会扑上来。
   他却笑了。脚跟已经踩实了。
```

**禁止：** 双方纯拼力量对轰，无思考、无战术、无预判。

---

## [后果] 代价可视化

**核心：** 不写"受伤了"，写后果。

```
❌ 他被砍了一刀，受了重伤。

✅ 他用左手接刀的时候就知道这手废了。
   血顺着手臂淌到肘弯，他握了握拳头——手指还能动，但使不上力。
```
